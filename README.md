# 🏁 Lidraughts Bot

Асинхронный бот для игры в русские шашки на платформе [lidraughts.org](https://lidraughts.org/) с использованием внешнего движка **Scan**.

> ⚠️ **Внимание:** Бот предназначен для образовательных целей. Использование ботов на игровых сайтах может нарушать их правила. Автор не несёт ответственности за последствия.

## 📦 Требования

- Python 3.10 или выше
- Установленный Git (для клонирования)
- Движок **Scan** (скачивается отдельно)
- Браузер Chromium (устанавливается автоматически через Playwright)

## ⚙️ Установка# Шашечный бот для lidraughts.org

## Установка и запуск

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/s0ldmys0ul/lidraughts-bot.git
cd lidraughts-bot
```

### 2. Установите Python 3.10 или новее
Скачайте с [официального сайта](https://python.org).

### 3. Создайте виртуальное окружение
```bash
python -m venv venv
```

**Активируйте:**
- **Windows:**
  ```bash
  venv\Scripts\activate
  ```
- **Linux/Mac:**
  ```bash
  source venv/bin/activate
  ```

### 4. Установите зависимости
```bash
pip install -r requirements.txt
```

### 5. Установите браузер для Playwright
```bash
playwright install
```

### 6. Скачайте движок **Scan**

Движок **Scan** не входит в репозиторий. Его нужно скачать отдельно.

- **Официальный репозиторий:** [rhalbersma/scan](https://github.com/rhalbersma/scan) (зеркало оригинального движка Fabien Letouzey)
- **Скачайте исполняемый файл** `scan.exe` (для Windows) или `scan_linux`/`scan_mac` (для других ОС) из корня репозитория.
- **Распакуйте** (или просто поместите) файл в папку `scan_31/` внутри проекта.
- **Либо** укажите свой путь к движку в файле `.env` (переменные `SCAN_PATH` и `SCAN_DIR`).

> **Важно:** Для работы Scan также могут потребоваться файлы из папки `data/` (базы данных, битбазы) и файл `scan.ini` с настройками. Скопируйте их в ту же папку, где находится исполняемый файл.

### 7. Настройте окружение
Скопируйте файл-пример:
```bash
cp .env.example .env
```
Отредактируйте `.env` – укажите пути к движку и другие параметры.

**Пример `.env`:**
```
SCAN_PATH=./scan_31/scan.exe
SCAN_DIR=./scan_31
MOVE_TIME=0.3
LOG_LEVEL=INFO
HEADLESS=False
```

### 8. Запустите бота
```bash
python main.py
```

Бот откроет браузер и будет ждать, пока вы откроете игру на `lidraughts.org`.  
Выберите цвет (белые или чёрные) и начните партию.

## Доступные команды

В консоли во время работы бота можно вводить команды:

| Команда | Описание |
|---------|----------|
| `mt=0.2` или `mt 0.2` | Установить время на ход (в секундах) |
| `status` | Показать текущие настройки |
| `cls` / `clear` | Очистить консоль |
| `help` | Вывести справку |

## Как это работает

1. Бот открывает браузер через Playwright.
2. Внедряет JavaScript-мостик, который перехватывает ходы противника.
3. При получении хода вызывает движок Scan для расчёта ответа.
4. Отправляет ход обратно на сайт с человеческой задержкой.

Подробнее – в [коде проекта](https://github.com/s0ldmys0ul/lidraughts-bot).
